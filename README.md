ShieldAI is an AI-powered scam detection web application that helps users identify scams **before they strike**.  
It analyzes **text, URLs, emails, SMS messages, voice recordings, and images** to detect scam indicators in real time.

> Built with a clean SaaS UI, modern Tailwind theming, and AI-driven analysis.

---

## Features

-  **Multi-Input Scam Analysis**
  - Text
  - URLs
  - Emails
  - SMS
  - Voice (record or upload audio)
  - Images

-  **AI-Powered Detection**
  - Detects 10+ scam indicators
  - Identifies urgency, manipulation, phishing patterns, and fraud signals

-  **Modern UI / UX**
  - Clean white theme with electric indigo accents
  - Fully responsive design
  - Dark mode support

-  **Fast & Secure**
  - Instant analysis
  - No data stored by default
  - Privacy-focused design
